    El nivel BEST requiere clave de Google Maps

    Este nivel carga el globo fotorrealista de Google (Photorealistic 3D Tiles) en vivo. Necesita una clave de la Maps JavaScript API con la Map Tiles API habilitada.

    Añade la clave a la URL:
    ?key=TU_CLAVE_DE_GOOGLE_MAPS

    Alta de la clave: ver best/README.md.
