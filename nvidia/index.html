An open agent system for physical retail.

NVIDIA is bringing agentic AI to the physical world. Admira is the application layer that runs it in retail and DOOH — a four-layer agentic loop deployed on NVIDIA edge, from GPU inference in the cloud to a local model on the screen itself.

The hard part of physical AI isn't the model — it's the application layer. Admira is that layer for physical retail: an open agent system that turns a street of screens into a live, self-improving product. Four layers — Create, Operate, Manage, Measure — running on NVIDIA, from cloud GPUs down to a Jetson at the edge.

Agentic at the edge
Every screen is a node: a local model decides what to show, when, and for whom — without a round-trip to the cloud. Physical AI where the audience actually is.
Vision that measures
Multi-camera occupancy, audience by computer vision and heatmaps — the store senses itself and feeds that signal straight back into what gets created next.
Open, not walled
Admira is an open agent system: it composes the best models and runs them on standard NVIDIA edge hardware in the field — no lock-in, deployed today.

NVIDIA has argued that the future belongs to open agent systems for the physical world — Admira is one, purpose-built for retail and DOOH. See NVIDIA's case for open agent systems ↗
